# 🏠 Housing Price Prediction — Buenos Aires & New York

This project focuses on predicting real estate prices in Buenos Aires and New York using linear models with increasing feature complexity and regularization techniques.

The objective is to evaluate how exploratory data analysis, feature engineering and regularization impact model performance.

`src/`

- `models.py`  
  Implementation of Linear Regression, Ridge (L2), and Lasso (L1)

- `utils.py`  
  Helper functions for:
  - Learning curves  
  - Plotting functions  

- `metrics.py`
  Implementation of metrics, MSE, RMSE and MAE

- `preprocessing.py`  
  Exploratory Data Analysis functions

- `data_splitting.py`  
  Helper functions for:
  - Splitting data
  - Cross validation


---

`notebooks/`

- Main analysis notebooks  
- Model training and evaluation  
- Visualization of results (learning curves, predictions)

---

`data/`

Datasets used in the project:

- Training data
